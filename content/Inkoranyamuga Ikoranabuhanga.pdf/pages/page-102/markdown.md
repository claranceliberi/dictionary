INKORANYAMUGA YIKORANABUHANGA

akunda, ibiri ku rupapuro yaguriyeho, bityo gushaka amakuru yihariye nyayo kandi yakoreshwa biroroha.

**Indangabikorwa z'inkoranabuhanga** (indāangabikorwā z'īinkōranabūhaānga). Eng: Menu-driven software. Fr: Logiciel piloté par menu. NK: Ikoranabuhanga rya mudasobwa. SH: Porogaramu runaka ya mudasobwa ishoboza abayikoresha kubona ibikorwa bitandukanye batabanje kujya kubishaka ahabanza.

**Indangabukana** (inseseengurabukana). Eng: Intensity descriptor. Fr: Descripteur d'intensité. NK: Ikoranabuhanga rya mudasobwa. SH: Ishusho y'ikoranabuhanga igaragaza muri make imiterere y'ubukana bw'ahantu cyangwa ikintu kiri mu isusho. Ifata amakuru yerekeye urwego rw'ubukana muri ako gace, bigatuma arigoritime (algorithms) zigereranya kandi zigasesengura ibikubiye mu isusho.

**Indangabwoko bwa fishiye** (indāangabwōko bwaa fishiye). Eng: File extension. Fr: Extension de fichier. NK: Ikoranabuhanga rya mudasobwa. SH: Inyuguti ziba ku mpera y'izina rya dosiye zigaragaza ubwoko bwa fishiye n'uko porogaramu iyifungura.

**Indangacyiciro cy'akazi** (indaangacyiciro cy'ākazi). Eng: Session token. Fr: Jeton de session. NK: Ikoranabuhanga rya mudasobwa. SH: Indanga yihariye yahanzwe ikorwa iyo ukoresha mudasobwa agiye ku rubuga rwa murandasi cyangwa ku nkoranabuhanga, ikaba ituma urwungano (sistemu) rumenya rukanacunga ibikorwa by'ukoresha mudasobwa ku mapaji menshi cyangwa ku byo yasabye muri icyo gihe ari kuri mudasobwa.

**Indangafishiye** (indaangafishiye). Eng: File descriptor; descriptor. Fr: Descripteur de fichiers. NK: Ikoranabuhanga rya mudasobwa. SH: Indangakintu yihariye imbonezamikorere iha ifishi iyo ifunguwe bwa mbere, igatuma za gahunda zishobora gukorana n'amafishiye, urucomekero cyangwa ubundi buryo bwinjiza bukanasohora amakuru, hagamijwe ko imbonezamikorere igira icyo ikora ku mafishi.

**Indangagikoresho koranabuhanga** (indāangagikoreesho kōranabūhaānga). Eng: Hardware token. Fr: Jeton matériel. NK: Ikoranabuhanga rya mudasobwa. SH: Urwungano rw'ibanga ngendanwa rukoreshwa mu iyemeza ry'urwungano.
