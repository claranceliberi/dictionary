INKORANYAMUGA YIKORANABUHANGA

Ikoranabuhanga rya mudasobwa. SH: Ubushobozi bw’igikoresho cy’ikoranabuhanga bwo kubika amakuru mesnhi ashoboka no kuyatanga igihe akenewe.

**Ububiko bunini bw’amakuru** (ububiko bunini bw’amakuru). Eng: Character Large Object (CLOB). Fr: Objet de grande taille. NK: Ikoranabuhanga rya mudasobwa. SH: Ubwoko bw’amakuru bukoreshwa mu kubika amakuru menshi y’inyuguti.

**Ububiko bw’amakuru** (ububiko bw’amakuru). Eng: Data storage; data storing. Fr: Stockage des données. NK: Ikoranabuhanga rya mudasobwa. SH: Ibika ry’amafishiye ahantu hafite umutekano umuntu ashobora kugera ku buryo bworoshye kandi bwizewe.

**Ububiko bwa mudasobwa** (ububiko bwaa mudasobwa). Eng: Computer storage; storage. Fr: Stockage informatique. NK: Ikoranabuhanga rya mudasobwa. SH: Ububiko cyangwa igice cya sisitemu ya mudasobwa kibika amakuru na porogaramu kugira ngo bizakoreshwe mu gihe kizaza.

**Ububiko bwa porogarame nkabuzi** (ububiko bwa porogaraame nkabuzi). Eng: Startup folder. Fr: Dossier de démarrage. NK: Ikoranabuhanga rya mudasobwa. SH: Icyumba cyihariye cy’amakuru y’ibikoresho bya mudasobwa, kibika porogaramu zikoreshwa igihe cyose akoresheje Windows.

**Ububiko bwagutse** (ububiko bwaagutse). Eng: Volume. Fr: Volume. NK: Ikoranabuhanga rya mudasobwa. SH: Igikoresho cyihariye mbikamakuru, twavuga nk’imbikamakuru ihoraho, imbikamashusho, imbikamakuru ntaremya, kikarangwa no kugira urwungano rw’amafishiye mudasobwa ishobora kubona, rukayibwira uko amadosiye n’amafishiye bitunganyije muri icyo gikoresho.

**Ububiko bwo kuri murandasi** (ububiko bwo kuri murandasi). Eng: Cloud Storage; OneDrive. Fr: Stockage en nuage; OneDrive. NK: Ikoranabuhanga rya mudasobwa. SH: Uburyo bwo kubika amakuru yawe kuri murandasi.

**Ububiko koranabuhanga bw’agateganyo** (ububiko kōranabūhaānga bw’agatēganyo). HI: Imbikamakuru y’akanya gato (imbikamakuru y’akāanya gato). Eng: Digital memory buffer; Buffer
